### 安装
在开始之前，请运行以下命令安装依赖:
```shell
yarn
```

### 开发
如要调试应用程序，请运行以下命令：
```shell
yarn dev
```

### 构建
如要构建应用程序，请运行以下命令：
```shell
yarn build
```
